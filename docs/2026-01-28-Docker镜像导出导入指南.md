## 一、镜像导出

### 1.1 使用导出脚本（推荐）

项目提供了便捷的导出脚本：

```bash
# 导出所有镜像到 docker-images 目录
./scripts/export_images.sh

# 导出所有镜像到指定目录
./scripts/export_images.sh -a -o /tmp/images

# 仅导出应用镜像
./scripts/export_images.sh --app

# 仅导出文档读取器镜像
./scripts/export_images.sh --docreader

# 仅导出前端镜像
./scripts/export_images.sh --frontend
```

**导出位置**：默认导出到项目根目录下的 `docker-images/` 目录

**导出文件**：
- `weknora-app-latest.tar` - 应用镜像
- `weknora-docreader-latest.tar` - 文档读取器镜像
- `weknora-ui-latest.tar` - 前端镜像

### 1.2 导出前准备

在导出镜像之前，需要先构建镜像：

```bash
# 构建所有镜像
./scripts/build_images.sh

# 或使用 Makefile
make build-images
```

### 1.3 检查镜像是否存在

```bash
# 查看本地镜像列表
docker images | grep weknora

# 应该看到类似输出：
# wechatopenai/weknora-app        latest    xxx    xxx ago    xxx MB
# wechatopenai/weknora-docreader  latest    xxx    xxx ago    xxx MB
# wechatopenai/weknora-ui         latest    xxx    xxx ago    xxx MB
```

## 二、镜像导入

### 2.1 使用 Docker Load 导入

```bash
# 导入应用镜像
docker load -i docker-images/weknora-app-latest.tar

# 导入文档读取器镜像
docker load -i docker-images/weknora-docreader-latest.tar

# 导入前端镜像
docker load -i docker-images/weknora-ui-latest.tar
```

### 2.2 批量导入

```bash
# 导入所有镜像
for tar_file in docker-images/*.tar; do
    echo "导入镜像: $tar_file"
    docker load -i "$tar_file"
done
```

### 2.3 验证导入结果

```bash
# 查看导入的镜像
docker images | grep weknora

# 应该看到导入的镜像
```

## 三、使用脚本导出

### 3.1 脚本功能

`scripts/export_images.sh` 脚本提供以下功能：

- ✅ 自动检查 Docker 环境
- ✅ 检查镜像是否存在
- ✅ 创建导出目录
- ✅ 导出镜像为 tar 文件
- ✅ 显示文件大小
- ✅ 支持选择性导出（全部/单个）

### 3.2 脚本选项

| 选项 | 说明 |
|------|------|
| `-h, --help` | 显示帮助信息 |
| `-a, --all` | 导出所有镜像（默认） |
| `-p, --app` | 仅导出应用镜像 |
| `-d, --docreader` | 仅导出文档读取器镜像 |
| `-f, --frontend` | 仅导出前端镜像 |
| `-o, --output DIR` | 指定导出目录（默认: docker-images） |
| `-v, --version` | 显示版本信息 |

### 3.3 使用示例

```bash
# 1. 导出所有镜像（默认到 docker-images 目录）
./scripts/export_images.sh

# 2. 导出所有镜像到自定义目录
./scripts/export_images.sh -a -o /path/to/images

# 3. 仅导出应用镜像
./scripts/export_images.sh -p

# 4. 导出应用和前端镜像
./scripts/export_images.sh -p -f
```

## 四、手动导出导入

### 4.1 手动导出镜像

如果不想使用脚本，可以手动导出：

```bash
# 1. 创建导出目录
mkdir -p docker-images

# 2. 导出应用镜像
docker save -o docker-images/weknora-app-latest.tar wechatopenai/weknora-app:latest

# 3. 导出文档读取器镜像
docker save -o docker-images/weknora-docreader-latest.tar wechatopenai/weknora-docreader:latest

# 4. 导出前端镜像
docker save -o docker-images/weknora-ui-latest.tar wechatopenai/weknora-ui:latest
```

### 4.2 手动导入镜像

```bash
# 1. 导入应用镜像
docker load -i docker-images/weknora-app-latest.tar

# 2. 导入文档读取器镜像
docker load -i docker-images/weknora-docreader-latest.tar

# 3. 导入前端镜像
docker load -i docker-images/weknora-ui-latest.tar
```

### 4.3 压缩镜像文件（可选）

为了节省空间，可以压缩 tar 文件：

```bash
# 压缩镜像文件
gzip docker-images/weknora-app-latest.tar
gzip docker-images/weknora-docreader-latest.tar
gzip docker-images/weknora-ui-latest.tar

# 导入压缩的镜像文件（Docker 支持直接导入 .tar.gz）
docker load -i docker-images/weknora-app-latest.tar.gz
```

## 五、常见问题

### 5.1 镜像不存在错误

**问题**：导出时提示镜像不存在

```
ERROR 镜像不存在: wechatopenai/weknora-app:latest
```

**解决方案**：

1. 先构建镜像：
   ```bash
   ./scripts/build_images.sh
   ```

2. 检查镜像是否存在：
   ```bash
   docker images | grep weknora
   ```

### 5.2 权限不足错误

**问题**：执行脚本时提示权限不足

```
Permission denied: ./scripts/export_images.sh
```

**解决方案**：

```bash
# 添加执行权限
chmod +x scripts/export_images.sh
```

### 5.3 磁盘空间不足

**问题**：导出镜像时提示磁盘空间不足

**解决方案**：

1. 检查磁盘空间：
   ```bash
   df -h
   ```

2. 清理不需要的 Docker 资源：
   ```bash
   docker system prune -a
   ```

3. 导出到其他有足够空间的目录：
   ```bash
   ./scripts/export_images.sh -o /path/to/large/disk/images
   ```

### 5.4 导入后镜像标签丢失

**问题**：导入镜像后，镜像名称变成了 `<none>`

**解决方案**：

导入后重新打标签：

```bash
# 查看导入的镜像 ID
docker images

# 重新打标签
docker tag <IMAGE_ID> wechatopenai/weknora-app:latest
docker tag <IMAGE_ID> wechatopenai/weknora-docreader:latest
docker tag <IMAGE_ID> wechatopenai/weknora-ui:latest
```

### 5.5 跨平台镜像导出导入

**问题**：在不同架构（amd64/arm64）之间导出导入镜像

**注意事项**：

- Docker 镜像包含平台信息
- 导出时使用 `--platform` 参数指定平台
- 导入时确保目标平台兼容

**示例**：

```bash
# 导出特定平台的镜像
docker buildx build --platform linux/amd64 -t wechatopenai/weknora-app:latest .
docker save -o weknora-app-amd64.tar wechatopenai/weknora-app:latest
```

## 七、最佳实践

### 7.1 导出前检查

```bash
# 1. 检查镜像是否存在
docker images | grep weknora

# 2. 检查磁盘空间
df -h

# 3. 检查导出目录权限
ls -ld docker-images/
```

### 7.2 导出后验证

```bash
# 1. 检查导出文件是否存在
ls -lh docker-images/*.tar

# 2. 验证文件完整性（可选）
file docker-images/*.tar

# 3. 测试导入（可选，在测试环境）
docker load -i docker-images/weknora-app-latest.tar
```

### 7.3 压缩和传输

```bash
# 1. 压缩镜像文件
cd docker-images
tar -czf weknora-images.tar.gz *.tar

# 2. 传输到目标服务器
scp weknora-images.tar.gz user@target-server:/path/to/destination/

# 3. 在目标服务器解压
tar -xzf weknora-images.tar.gz

# 4. 导入镜像
docker load -i weknora-app-latest.tar
docker load -i weknora-docreader-latest.tar
docker load -i weknora-ui-latest.tar
```
