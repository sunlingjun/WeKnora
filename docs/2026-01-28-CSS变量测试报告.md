### 1. 暗色模式字体颜色变量修复

**修改内容**：
- 将暗色模式下的 `--td-font-gray-*` 变量映射到 `--td-font-white-*`
- 确保直接使用 `--td-font-gray-*` 的地方在暗色模式下正常显示

**修改位置**：
```css

    /* 暗色模式下字体颜色变量映射到白色 */
    --td-font-gray-1: var(--td-font-white-1);
    --td-font-gray-2: var(--td-font-white-2);
    --td-font-gray-3: var(--td-font-white-3);
    --td-font-gray-4: var(--td-font-white-4);

}
```

## ⚠️ 注意事项

1. **变量优先级**：CSS 变量遵循级联规则，暗色模式的变量会覆盖亮色模式
2. **浏览器兼容性**：CSS 变量需要现代浏览器支持（IE11 不支持）
3. **回退值**：建议在使用时提供回退值，例如：`var(--td-info-color, #0052d9)`
