## 概述

### CAS 简介

**CAS (Central Authentication Service)** 是由耶鲁大学开发的企业级单点登录解决方案，广泛应用于高校、科研机构和大型企业。

### 核心特点

- ✅ **简单易用**：协议简单，易于集成
- ✅ **广泛支持**：高校和科研机构常用
- ✅ **代理认证**：支持代理票据（Proxy Ticket）
- ✅ **多版本兼容**：支持 CAS 1.0、2.0、3.0
- ✅ **属性传递**：支持用户属性传递（CAS 2.0+）

### 适用场景

- ✅ 高校统一身份认证系统
- ✅ 科研机构多系统集成
- ✅ 企业内部多应用 SSO
- ✅ 已有 CAS 服务器的组织

## CAS 协议详解

### CAS 认证流程

#### 1. 服务票据（Service Ticket）流程（CAS 1.0/2.0/3.0）

```
┌─────────┐                    ┌─────────┐                    ┌─────────┐
│  用户   │                    │ WeKnora│                    │CAS服务器│
│ 浏览器  │                    │ (SP)    │                    │ (IdP)   │
└────┬────┘                    └────┬────┘                    └────┬────┘
     │                              │                              │
     │ 1. 访问受保护资源             │                              │
     ├─────────────────────────────>│                              │
     │                              │                              │
     │                              │ 2. 重定向到 CAS 登录页       │
     │                              │    GET /cas/login?service=...│
     │<─────────────────────────────┤                              │
     │                              │                              │
     │ 3. 重定向到 CAS               │                              │
     ├─────────────────────────────────────────────────────────────>│
     │                              │                              │
     │ 4. 用户在 CAS 页面登录        │                              │
     │                              │                              │
     │ 5. CAS 验证成功，重定向回 SP   │                              │
     │    GET /cas/callback?ticket=ST-xxx                          │
     │<─────────────────────────────────────────────────────────────┤
     │                              │                              │
     │                              │ 6. 验证服务票据               │
     │                              │    GET /cas/serviceValidate?  │
     │                              │    service=...&ticket=ST-xxx  │
     │                              ├─────────────────────────────>│
     │                              │                              │
     │                              │ 7. 返回用户信息（XML）        │
     │                              │<─────────────────────────────┤
     │                              │                              │
     │                              │ 8. 创建/更新本地用户          │
     │                              │ 9. 生成 WeKnora JWT Token    │
     │                              │                              │
     │ 10. 重定向到前端（带 Token）  │                              │
     │<─────────────────────────────┤                              │
     │                              │                              │
```

#### 2. 代理票据（Proxy Ticket）流程（CAS 2.0+）

```
┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐
│  用户   │  │ WeKnora │  │CAS服务器│  │后端服务 │
│ 浏览器  │  │  (SP)   │  │  (IdP)  │  │  (Proxy)│
└────┬────┘  └────┬────┘  └────┬────┘  └────┬────┘
     │           │            │            │
     │ 1. 登录获取 PTG        │            │
     ├──────────>│            │            │
     │           ├───────────>│            │
     │           │<───────────┤            │
     │           │            │            │
     │ 2. 使用 PT 调用后端服务 │            │
     │<──────────┤            │            │
     │           ├────────────────────────>│
     │           │            │            │
     │           │            │ 3. 验证 PT │
     │           │<───────────┤            │
     │           │            │            │
     │           │ 4. 返回结果 │            │
     │<──────────┤<────────────────────────┤
```

## CAS 版本说明

### CAS 1.0

**特点**：
- 最简单的版本
- 只支持服务票据（Service Ticket）
- 验证返回简单的用户名

**验证请求**：
```
GET /cas/serviceValidate?service={service}&ticket={ticket}
```

**验证响应**：
```xml
<cas:serviceResponse xmlns:cas="http://www.yale.edu/tp/cas">
    <cas:authenticationSuccess>
        <cas:user>username</cas:user>
    </cas:authenticationSuccess>
</cas:serviceResponse>
```

### CAS 2.0

**特点**：
- 支持代理票据（Proxy Ticket）
- 支持用户属性传递
- XML 格式响应

**验证请求**：
```
GET /cas/serviceValidate?service={service}&ticket={ticket}&format=XML
GET /cas/proxyValidate?service={service}&ticket={ticket}&pgtUrl={pgtUrl}
```

**验证响应**：
```xml
<cas:serviceResponse xmlns:cas="http://www.yale.edu/tp/cas">
    <cas:authenticationSuccess>
        <cas:user>username</cas:user>
        <cas:attributes>
            <cas:email>user@example.com</cas:email>
            <cas:displayName>John Doe</cas:displayName>
            <cas:department>IT</cas:department>
        </cas:attributes>
        <cas:proxyGrantingTicket>PGT-xxx</cas:proxyGrantingTicket>
    </cas:authenticationSuccess>
</cas:serviceResponse>
```

### CAS 3.0

**特点**：
- 支持 JSON 格式响应
- RESTful API
- 更灵活的属性传递

**验证请求**：
```
GET /cas/p3/serviceValidate?service={service}&ticket={ticket}&format=JSON
```

**验证响应（JSON）**：
```json
{
  "serviceResponse": {
    "authenticationSuccess": {
      "user": "username",
      "attributes": {
        "email": "user@example.com",
        "displayName": "John Doe",
        "department": "IT"
      },
      "proxyGrantingTicket": "PGT-xxx"
    }
  }
}
```

### 2. CAS Provider 实现

```go
package provider

import (
    "context"
    "encoding/json"
    "encoding/xml"
    "fmt"
    "io"
    "net/http"
    "net/url"
    "strings"
    "time"

    "github.com/Tencent/WeKnora/internal/logger"
)

// CASProvider CAS 认证提供商
type CASProvider struct {
    providerID   string
    name         string
    serverURL    string
    version      string // "1.0", "2.0", "3.0"
    serviceURL   string
    validateURL  string
    loginURL     string
    logoutURL    string
    httpClient   *http.Client
    config       map[string]interface{}
    attrMapping  map[string]string
}

// CASConfig CAS 配置
type CASConfig struct {
    ServerURL        string            `json:"server_url"`
    Version          string            `json:"version"` // "1.0", "2.0", "3.0"
    ServiceURL       string            `json:"service_url"`
    ValidateURL      string            `json:"validate_url"` // 可选，默认根据版本自动选择
    LoginURL         string            `json:"login_url"`     // 可选，默认 "/login"
    LogoutURL        string            `json:"logout_url"`    // 可选，默认 "/logout"
    ProxyCallbackURL string            `json:"proxy_callback_url"`
    AcceptAnyProxy   bool              `json:"accept_any_proxy"`
    Renew            bool              `json:"renew"`
    Gateway          bool              `json:"gateway"`
    AttributeMapping map[string]string `json:"attribute_mapping"`
}

// NewCASProvider 创建 CAS Provider
func NewCASProvider(providerID, name string, config map[string]interface{}) (*CASProvider, error) {
    // 解析配置
    configJSON, err := json.Marshal(config)
    if err != nil {
        return nil, fmt.Errorf("failed to marshal config: %w", err)
    }

    var casConfig CASConfig
    if err := json.Unmarshal(configJSON, &casConfig); err != nil {
        return nil, fmt.Errorf("failed to unmarshal config: %w", err)
    }

    // 设置默认值
    if casConfig.Version == "" {
        casConfig.Version = "3.0"
    }
    if casConfig.LoginURL == "" {
        casConfig.LoginURL = "/login"
    }
    if casConfig.LogoutURL == "" {
        casConfig.LogoutURL = "/logout"
    }
    if casConfig.ValidateURL == "" {
        // 根据版本选择默认验证端点
        switch casConfig.Version {
        case "1.0", "2.0":
            casConfig.ValidateURL = "/serviceValidate"
        case "3.0":
            casConfig.ValidateURL = "/p3/serviceValidate"
        default:
            casConfig.ValidateURL = "/p3/serviceValidate"
        }
    }

    // 解析属性映射
    attrMapping := make(map[string]string)
    if casConfig.AttributeMapping != nil {
        attrMapping = casConfig.AttributeMapping
    } else {
        // 默认映射
        attrMapping = map[string]string{
            "username":     "user",
            "email":        "email",
            "display_name": "displayName",
        }
    }

    return &CASProvider{
        providerID:  providerID,
        name:        name,
        serverURL:   casConfig.ServerURL,
        version:     casConfig.Version,
        serviceURL:  casConfig.ServiceURL,
        validateURL: casConfig.ValidateURL,
        loginURL:    casConfig.LoginURL,
        logoutURL:   casConfig.LogoutURL,
        httpClient: &http.Client{
            Timeout: 30 * time.Second,
        },
        config:      config,
        attrMapping: attrMapping,
    }, nil
}

// GetProviderID 返回提供商 ID
func (p *CASProvider) GetProviderID() string {
    return p.providerID
}

// GetProviderType 返回提供商类型
func (p *CASProvider) GetProviderType() string {
    return "cas"
}

// GetAuthorizationURL 生成 CAS 登录 URL
func (p *CASProvider) GetAuthorizationURL(ctx context.Context, state string, redirectURI string) (string, error) {
    baseURL := strings.TrimSuffix(p.serverURL, "/")
    loginPath := p.loginURL
    if !strings.HasPrefix(loginPath, "/") {
        loginPath = "/" + loginPath
    }

    u, err := url.Parse(baseURL + loginPath)
    if err != nil {
        return "", fmt.Errorf("failed to parse login URL: %w", err)
    }

    q := u.Query()
    q.Set("service", redirectURI)
    
    // CAS 3.0 支持 renew 和 gateway 参数
    if p.version == "3.0" {
        // renew: 强制重新登录
        // gateway: 静默登录（如果已登录则返回，否则不提示登录）
    }

    u.RawQuery = q.Encode()
    return u.String(), nil
}

// ValidateTicket 验证服务票据
func (p *CASProvider) ValidateTicket(ctx context.Context, ticket string, serviceURL string) (*UserInfo, error) {
    baseURL := strings.TrimSuffix(p.serverURL, "/")
    validatePath := p.validateURL
    if !strings.HasPrefix(validatePath, "/") {
        validatePath = "/" + validatePath
    }

    u, err := url.Parse(baseURL + validatePath)
    if err != nil {
        return nil, fmt.Errorf("failed to parse validate URL: %w", err)
    }

    q := u.Query()
    q.Set("service", serviceURL)
    q.Set("ticket", ticket)
    
    // CAS 3.0 支持 format 参数
    if p.version == "3.0" {
        q.Set("format", "JSON")
    }

    u.RawQuery = q.Encode()

    logger.Infof(ctx, "Validating CAS ticket: %s", u.String())

    req, err := http.NewRequestWithContext(ctx, "GET", u.String(), nil)
    if err != nil {
        return nil, fmt.Errorf("failed to create request: %w", err)
    }

    resp, err := p.httpClient.Do(req)
    if err != nil {
        return nil, fmt.Errorf("failed to validate ticket: %w", err)
    }
    defer resp.Body.Close()

    if resp.StatusCode != http.StatusOK {
        return nil, fmt.Errorf("CAS validation failed with status: %d", resp.StatusCode)
    }

    body, err := io.ReadAll(resp.Body)
    if err != nil {
        return nil, fmt.Errorf("failed to read response: %w", err)
    }

    // 根据版本和格式解析响应
    if p.version == "3.0" {
        return p.parseJSONResponse(ctx, body)
    } else {
        return p.parseXMLResponse(ctx, body)
    }
}

// parseJSONResponse 解析 CAS 3.0 JSON 响应
func (p *CASProvider) parseJSONResponse(ctx context.Context, body []byte) (*UserInfo, error) {
    var response struct {
        ServiceResponse struct {
            AuthenticationSuccess *struct {
                User       string                 `json:"user"`
                Attributes map[string]interface{} `json:"attributes"`
            } `json:"authenticationSuccess"`
            AuthenticationFailure *struct {
                Code    string `json:"code"`
                Message string `json:"message"`
            } `json:"authenticationFailure"`
        } `json:"serviceResponse"`
    }

    if err := json.Unmarshal(body, &response); err != nil {
        return nil, fmt.Errorf("failed to parse JSON response: %w", err)
    }

    if response.ServiceResponse.AuthenticationFailure != nil {
        failure := response.ServiceResponse.AuthenticationFailure
        return nil, fmt.Errorf("CAS authentication failed: %s - %s", failure.Code, failure.Message)
    }

    if response.ServiceResponse.AuthenticationSuccess == nil {
        return nil, fmt.Errorf("invalid CAS response: no authentication result")
    }

    success := response.ServiceResponse.AuthenticationSuccess
    return p.mapUserInfo(success.User, success.Attributes), nil
}

// parseXMLResponse 解析 CAS 1.0/2.0 XML 响应
func (p *CASProvider) parseXMLResponse(ctx context.Context, body []byte) (*UserInfo, error) {
    var response struct {
        XMLName xml.Name `xml:"serviceResponse"`
        Success *struct {
            User       string `xml:"user"`
            Attributes struct {
                Attributes []struct {
                    Name  string `xml:"name,attr"`
                    Value string `xml:",chardata"`
                } `xml:"attribute"`
            } `xml:"attributes"`
        } `xml:"authenticationSuccess"`
        Failure *struct {
            Code    string `xml:"code,attr"`
            Message string `xml:",chardata"`
        } `xml:"authenticationFailure"`
    }

    if err := xml.Unmarshal(body, &response); err != nil {
        return nil, fmt.Errorf("failed to parse XML response: %w", err)
    }

    if response.Failure != nil {
        return nil, fmt.Errorf("CAS authentication failed: %s - %s", response.Failure.Code, response.Failure.Message)
    }

    if response.Success == nil {
        return nil, fmt.Errorf("invalid CAS response: no authentication result")
    }

    // 将 XML 属性转换为 map
    attrs := make(map[string]interface{})
    for _, attr := range response.Success.Attributes.Attributes {
        attrs[attr.Name] = attr.Value
    }

    return p.mapUserInfo(response.Success.User, attrs), nil
}

// mapUserInfo 映射用户信息
func (p *CASProvider) mapUserInfo(username string, attributes map[string]interface{}) *UserInfo {
    userInfo := &UserInfo{
        ExternalUserID: username,
        Username:       username,
        Attributes:     make(map[string]string),
    }

    // 映射属性
    for key, value := range attributes {
        if strValue, ok := value.(string); ok {
            userInfo.Attributes[key] = strValue
        } else if arrValue, ok := value.([]interface{}); ok && len(arrValue) > 0 {
            // CAS 属性可能是数组，取第一个值
            if strValue, ok := arrValue[0].(string); ok {
                userInfo.Attributes[key] = strValue
            }
        }
    }

    // 根据映射配置提取字段
    if emailKey := p.attrMapping["email"]; emailKey != "" {
        if email, ok := userInfo.Attributes[emailKey]; ok {
            userInfo.Email = email
        }
    }
    if displayNameKey := p.attrMapping["display_name"]; displayNameKey != "" {
        if displayName, ok := userInfo.Attributes[displayNameKey]; ok {
            userInfo.DisplayName = displayName
        }
    }
    if avatarKey := p.attrMapping["avatar"]; avatarKey != "" {
        if avatar, ok := userInfo.Attributes[avatarKey]; ok {
            userInfo.Avatar = avatar
        }
    }

    // 如果没有映射到邮箱，使用用户名
    if userInfo.Email == "" {
        userInfo.Email = username
    }

    return userInfo
}

// GetUserInfo 获取用户信息（通过票据）
func (p *CASProvider) GetUserInfo(ctx context.Context, ticket string, serviceURL string) (*UserInfo, error) {
    return p.ValidateTicket(ctx, ticket, serviceURL)
}
```

```go
package service

import (
    "context"
    "fmt"
    "time"

    "github.com/google/uuid"

// CASService CAS 认证服务
type CASService struct {
    provider        provider.AuthProvider
    userService     interfaces.UserService
    tenantService   interfaces.TenantService
    externalAuthRepo interfaces.ExternalAuthRepository
}

// NewCASService 创建 CAS 服务
func NewCASService(
    casProvider provider.AuthProvider,
    userService interfaces.UserService,
    tenantService interfaces.TenantService,
    externalAuthRepo interfaces.ExternalAuthRepository,
) *CASService {
    return &CASService{
        provider:        casProvider,
        userService:     userService,
        tenantService:   tenantService,
        externalAuthRepo: externalAuthRepo,
    }
}

// HandleCallback 处理 CAS 回调
func (s *CASService) HandleCallback(ctx context.Context, ticket, serviceURL string, tenantID uint64) (*types.LoginResponse, error) {
    logger.Info(ctx, "Handling CAS callback")

    // 1. 验证票据
    userInfo, err := s.provider.ValidateTicket(ctx, ticket, serviceURL)
    if err != nil {
        logger.Errorf(ctx, "Failed to validate CAS ticket: %v", err)
        return nil, fmt.Errorf("ticket validation failed: %w", err)
    }

    logger.Infof(ctx, "CAS ticket validated for user: %s", userInfo.ExternalUserID)

    // 2. 查找或创建用户
    user, err := s.findOrCreateUser(ctx, userInfo, tenantID)
    if err != nil {
        logger.Errorf(ctx, "Failed to find or create user: %v", err)
        return nil, fmt.Errorf("user creation failed: %w", err)
    }

    // 3. 关联外部账号
    providerID := s.provider.GetProviderID()
    err = s.linkExternalAccount(ctx, user.ID, providerID, userInfo)
    if err != nil {
        logger.Warnf(ctx, "Failed to link external account: %v", err)
        // 不阻断登录流程
    }

    // 4. 生成 WeKnora JWT Token
    accessToken, refreshToken, err := s.userService.GenerateTokens(ctx, user)
    if err != nil {
        logger.Errorf(ctx, "Failed to generate tokens: %v", err)
        return nil, fmt.Errorf("token generation failed: %w", err)
    }

    // 5. 获取租户信息
    tenant, err := s.tenantService.GetTenantByID(ctx, user.TenantID)
    if err != nil {
        logger.Warnf(ctx, "Failed to get tenant: %v", err)
    }

    return &types.LoginResponse{
        Success:      true,
        Message:      "CAS login successful",
        User:         user,
        Tenant:       tenant,
        Token:        accessToken,
        RefreshToken: refreshToken,
    }, nil
}

// findOrCreateUser 查找或创建用户
func (s *CASService) findOrCreateUser(ctx context.Context, userInfo *provider.UserInfo, tenantID uint64) (*types.User, error) {
    providerID := s.provider.GetProviderID()

    // 1. 尝试通过外部账号关联查找
    externalAccount, err := s.externalAuthRepo.GetExternalAccount(ctx, providerID, userInfo.ExternalUserID)
    if err == nil && externalAccount != nil {
        user, err := s.userService.GetUserByID(ctx, externalAccount.UserID)
        if err == nil && user != nil {
            logger.Infof(ctx, "Found existing user via external account: %s", user.ID)
            return user, nil
        }
    }

    // 2. 尝试通过邮箱查找
    if userInfo.Email != "" {
        user, err := s.userService.GetUserByEmail(ctx, userInfo.Email)
        if err == nil && user != nil {
            logger.Infof(ctx, "Found existing user via email: %s", user.ID)
            return user, nil
        }
    }

    // 3. 创建新用户
    logger.Info(ctx, "Creating new user from CAS")
    
    // 生成用户名（如果邮箱可用，使用邮箱前缀；否则使用外部用户ID）
    username := userInfo.Username
    if username == "" {
        if userInfo.Email != "" {
            // 提取邮箱前缀作为用户名
            username = strings.Split(userInfo.Email, "@")[0]
        } else {
            username = userInfo.ExternalUserID
        }
    }

    // 确保用户名唯一
    baseUsername := username
    for i := 1; i <= 10; i++ {
        existingUser, _ := s.userService.GetUserByUsername(ctx, username)
        if existingUser == nil {
            break
        }
        username = fmt.Sprintf("%s%d", baseUsername, i)
    }

    // 创建租户（如果需要）
    if tenantID == 0 {
        tenant := &types.Tenant{
            Name:        fmt.Sprintf("%s's Workspace", username),
            Description: "Default workspace",
            Status:      "active",
        }
        createdTenant, err := s.tenantService.CreateTenant(ctx, tenant)
        if err != nil {
            return nil, fmt.Errorf("failed to create tenant: %w", err)
        }
        tenantID = createdTenant.ID
    }

    // 创建用户
    user := &types.User{
        ID:               uuid.New().String(),
        Username:         username,
        Email:            userInfo.Email,
        PasswordHash:     "", // CAS 用户不需要密码
        Avatar:           userInfo.Avatar,
        TenantID:         tenantID,
        IsActive:         true,
        AuthSource:       "cas",
        PasswordRequired: false,
        CreatedAt:        time.Now(),
        UpdatedAt:        time.Now(),
    }

    // 这里需要调用 repository 创建用户，因为 userService.Register 需要密码
    // 需要添加一个 CreateUserWithoutPassword 方法
    // 暂时使用 Register 但设置空密码（需要修改 Register 方法支持）
    // 或者直接调用 repository
    
    // 简化处理：调用 repository
    // 注意：这里需要访问 repository，可能需要重构
    // 暂时返回错误，提示需要实现 CreateUserWithoutPassword
    return nil, fmt.Errorf("user creation not implemented yet, need CreateUserWithoutPassword method")
}

// linkExternalAccount 关联外部账号
func (s *CASService) linkExternalAccount(ctx context.Context, userID, providerID string, userInfo *provider.UserInfo) error {
    externalAccount := &types.UserExternalAccount{
        ID:               uuid.New().String(),
        UserID:           userID,
        ProviderID:       providerID,
```

```go
package handler

import (

// CASHandler CAS 认证处理器
type CASHandler struct {
    casService    *service.CASService

}

// NewCASHandler 创建 CAS Handler
func NewCASHandler(casService *service.CASService, tenantService interfaces.TenantService) *CASHandler {
    return &CASHandler{
        casService:    casService,
        tenantService: tenantService,
    }
}

// Authorize 重定向到 CAS 登录页
// @Summary      CAS 登录授权
// @Description  重定向到 CAS 服务器登录页面
// @Tags         认证
// @Accept       json
// @Produce      json
// @Param        provider_id  path      string  true  "CAS 提供商 ID"
// @Success      302          {string}  string  "重定向到 CAS 登录页"
// @Failure      400          {object}  errors.AppError  "请求参数错误"
// @Router       /auth/external/{provider_id}/cas/authorize [get]
func (h *CASHandler) Authorize(c *gin.Context) {
    ctx := c.Request.Context()
    providerID := c.Param("provider_id")

    // 获取租户 ID（从 Header 或 Query）
    tenantID := h.getTenantID(c)

    // 构建回调 URL
    callbackURL := h.buildCallbackURL(c, providerID)
    
    // 生成授权 URL
    authURL, err := h.casService.GetAuthorizationURL(ctx, "", callbackURL)
    if err != nil {
        logger.Errorf(ctx, "Failed to generate authorization URL: %v", err)
        c.JSON(http.StatusBadRequest, gin.H{
            "error": "Failed to generate authorization URL",
        })
        return
    }

    // 存储 state 到 session（可选，CAS 不需要 state，但可以用于跟踪）
    // 这里简化处理，直接重定向

    logger.Infof(ctx, "Redirecting to CAS: %s", authURL)
    c.Redirect(http.StatusFound, authURL)
}

// Callback 处理 CAS 回调
// @Summary      CAS 登录回调
// @Description  处理 CAS 服务器回调，验证票据并登录
// @Tags         认证
// @Accept       json
// @Produce      json
// @Param        provider_id  path      string  true  "CAS 提供商 ID"
// @Param        ticket       query     string  true  "CAS 服务票据"
// @Success      302          {string}  string  "重定向到前端（带 Token）"
// @Failure      400          {object}  errors.AppError  "请求参数错误"
// @Failure      401          {object}  errors.AppError  "认证失败"
// @Router       /auth/external/{provider_id}/cas/callback [get]
func (h *CASHandler) Callback(c *gin.Context) {
    ctx := c.Request.Context()
    providerID := c.Param("provider_id")
    ticket := c.Query("ticket")

    if ticket == "" {
        logger.Error(ctx, "Missing ticket parameter")
        c.JSON(http.StatusBadRequest, gin.H{
            "error": "Missing ticket parameter",
        })
        return
    }

    // 获取租户 ID
    tenantID := h.getTenantID(c)

    // 构建服务 URL（回调 URL）
    serviceURL := h.buildCallbackURL(c, providerID)

    // 处理回调
    loginResponse, err := h.casService.HandleCallback(ctx, ticket, serviceURL, tenantID)
    if err != nil {
        logger.Errorf(ctx, "CAS callback failed: %v", err)
        c.JSON(http.StatusUnauthorized, gin.H{
            "error": "CAS authentication failed",
            "message": err.Error(),
        })
        return
    }

    // 重定向到前端，携带 Token
    frontendURL := h.getFrontendURL(c)
    redirectURL := frontendURL + "/auth/callback?token=" + url.QueryEscape(loginResponse.Token)
    
    if loginResponse.RefreshToken != "" {
        redirectURL += "&refresh_token=" + url.QueryEscape(loginResponse.RefreshToken)
    }

    logger.Infof(ctx, "CAS login successful, redirecting to frontend")
    c.Redirect(http.StatusFound, redirectURL)
}

// getTenantID 获取租户 ID
func (h *CASHandler) getTenantID(c *gin.Context) uint64 {
    // 从 Header 获取
    if tenantHeader := c.GetHeader("X-Tenant-ID"); tenantHeader != "" {
        if tenantID, err := strconv.ParseUint(tenantHeader, 10, 64); err == nil {
            return tenantID
        }
    }
    
    // 从 Query 获取
    if tenantQuery := c.Query("tenant_id"); tenantQuery != "" {
        if tenantID, err := strconv.ParseUint(tenantQuery, 10, 64); err == nil {
            return tenantID
        }
    }
    
    return 0
}

// buildCallbackURL 构建回调 URL
func (h *CASHandler) buildCallbackURL(c *gin.Context, providerID string) string {
    scheme := "https"
    if c.GetHeader("X-Forwarded-Proto") != "" {
        scheme = c.GetHeader("X-Forwarded-Proto")
    } else if c.Request.TLS == nil {
        scheme = "http"
    }

    host := c.GetHeader("X-Forwarded-Host")
    if host == "" {
        host = c.Request.Host
    }

    return fmt.Sprintf("%s://%s/api/v1/auth/external/%s/cas/callback", scheme, host, providerID)
}

// getFrontendURL 获取前端 URL
func (h *CASHandler) getFrontendURL(c *gin.Context) string {
    // 从配置或环境变量获取
    frontendURL := os.Getenv("FRONTEND_URL")
    if frontendURL == "" {
        frontendURL = "http://localhost:3000"
    }
    return frontendURL
}
```

## 常见问题

### 1. 票据验证失败

**问题**：`CAS authentication failed: INVALID_TICKET`

**原因**：
- 票据已过期（通常 5-10 秒）
- Service URL 不匹配
- 票据已被使用

**解决**：
- 检查 Service URL 配置是否一致
- 确保回调 URL 与 CAS 服务器注册的一致
- 检查票据是否重复使用

### 2. 属性映射失败

**问题**：用户信息中缺少邮箱或其他属性

**原因**：
- CAS 服务器未释放属性
- 属性映射配置错误

**解决**：
- 检查 CAS 服务器属性释放策略
- 验证 `attribute_mapping` 配置
- 查看 CAS 响应中的实际属性名称

### 3. 用户创建失败

**问题**：`user creation not implemented yet`

**原因**：
- `CreateUserWithoutPassword` 方法未实现

**解决**：
- 在 `UserService` 中添加 `CreateUserWithoutPassword` 方法
- 或修改 `Register` 方法支持空密码

### 4. 重定向循环

**问题**：CAS 登录后无限重定向

**原因**：
- 前端回调 URL 配置错误
- Token 验证失败

**解决**：
- 检查 `FRONTEND_URL` 环境变量
- 验证 Token 生成和验证逻辑
- 检查前端路由配置
