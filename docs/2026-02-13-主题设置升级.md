# 主题设置升级

> **文档创建**：2026-02-13  
> **功能**：常规设置中增加外观主题（浅色/深色/跟随系统），支持应用启动时恢复

---

## 一、升级概述

| 项目 | 说明 |
|------|------|
| **升级前** | 无主题切换入口，依赖系统或默认样式 |
| **升级后** | 在设置 → 常规 中可切换浅色、深色或跟随系统，并持久化保存 |
| **依赖** | `theme.css` 已定义 `:root[theme-mode="light"]` 与 `:root[theme-mode="dark"]` 变量 |

---

## 二、修改点汇总

| 层级 | 文件 | 修改内容 |
|------|------|----------|
| 入口 | `frontend/src/main.ts` | 应用启动时从 localStorage 读取主题并设置 `document.documentElement.setAttribute('theme-mode', ...)` |
| 组件 | `frontend/src/views/settings/GeneralSettings.vue` | 新增外观主题选择行（t-select：light/dark/system），监听系统偏好变化 |
| i18n | `frontend/src/i18n/locales/zh-CN.ts`、`en-US.ts`、`ru-RU.ts`、`ko-KR.ts` | 新增 `general.appearance` 下 theme、themeDescription、themeLight、themeDark、themeSystem、themeSaved |

---

## 三、实现要点

### 3.1 主题模式

| 值 | 说明 |
|----|------|
| `light` | 浅色模式 |
| `dark` | 深色模式 |
| `system` | 跟随系统（`prefers-color-scheme`） |

### 3.2 持久化

- **存储 key**：`WeKnora_general_settings`
- **结构**：`{ language: string, theme: 'light' | 'dark' | 'system' }`

### 3.3 应用时机

1. **启动**：`main.ts` 中在挂载 App 前执行，确保首屏即应用主题
2. **用户切换**：`GeneralSettings` 中 `handleThemeChange` 立即调用 `applyTheme` 并保存
3. **跟随系统**：当选择 system 时，监听 `matchMedia('(prefers-color-scheme: dark)').change`，系统切换时自动更新

### 3.4 应用方式

```javascript
document.documentElement.setAttribute('theme-mode', actual)
// actual 为 'light' 或 'dark'，由 theme.css 的 :root[theme-mode="light"] / :root[theme-mode="dark"] 生效
```
