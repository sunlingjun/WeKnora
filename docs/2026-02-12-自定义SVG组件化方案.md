# 自定义 SVG 组件化方案

> **文档创建**：2026-02-05

## 一、目标

1. **统一管理**：自定义 SVG 图标组件化，各页面统一引用
2. **主题适配**：支持自定义颜色，可跟随 TDesign 主题切换
3. **消除冗余**：替代多份同图标不同色版本（如 `xxx.svg`、`xxx-green.svg`、`xxx-grey.svg`）

---

## 二、现状

| 类型 | 示例 | 问题 |
|------|------|------|
| currentColor 型 | organization.svg, user.svg | ✅ 可继承父级颜色 |
| 硬编码色型 | zhishiku.svg, zhishiku-green.svg | ❌ 无法跟随主题 |

---

## 三、方案设计

### 3.1 架构

```
src/components/icons/
├── index.ts           # 统一导出
├── SvgIcon.vue        # 通用图标容器
└── registry.ts        # 图标注册表（zhishiku/organization/ziliao/agent/user）
```

### 3.2 SvgIcon 组件

| 属性 | 类型 | 说明 |
|------|------|------|
| name | `string` | 必填，图标名称 |
| size | `number \| string` | 默认 20，单位 px |
| color | `string` | 自定义颜色 |
| theme | `'default' \| 'brand' \| 'secondary' \| 'placeholder' \| 'anti'` | 主题预设 |

**theme 映射**：default → `--td-text-color-primary`，brand → `--td-brand-color`，secondary → `--td-text-color-secondary`，placeholder → `--td-text-color-placeholder`，anti → `--td-text-color-anti`。

**用法**：
```vue
<SvgIcon name="organization" theme="brand" :size="20" />
<SvgIcon name="zhishiku" color="var(--td-brand-color)" :size="24" />
<span><SvgIcon name="ziliao" :size="16" /> 资料</span>
```

### 3.3 SVG 规范化

将 `fill` / `stroke` 改为 `currentColor`，组件通过 `color` 或外层 CSS 控制颜色。

---

## 四、实施步骤

1. **Phase 1**：创建 SvgIcon.vue、图标注册表，规范化 SVG
2. **Phase 2**：替换 `<img src="xxx.svg">` 为 `<SvgIcon name="xxx" />`，删除冗余变体
3. **Phase 3**：补充其余图标，支持 `aria-label`、`role="img"`

---

**结论**：采用内联 Vue 组件 + currentColor，实现简单、主题支持完善、易维护。
