# 图标主题适配升级

> **文档创建**：2026-02-13  
> **功能**：将硬编码颜色的 SVG 图标改为使用 `currentColor` + CSS 变量，实现浅色/深色主题自适应

---

## 一、升级概述

| 项目 | 说明 |
|------|------|
| **问题** | 部分图标（如 at-icon.svg）使用固定色 `#00000099`，在暗色主题下仍为黑色，难以辨认 |
| **方案** | 使用内联 SVG + `stroke="currentColor"` / `fill="currentColor"`，通过父元素 `color` 继承主题变量 |
| **参考** | [2026-02-05-颜色统一化改造](./2026-02-05-颜色统一化改造.md)、[2026-02-12-自定义SVG组件化方案](./2026-02-12-自定义SVG组件化方案.md) |

---

## 二、适配原则

### 2.1 推荐写法

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 内联 SVG + `currentColor` | `stroke="currentColor"` 或 `fill="currentColor"`，由父级 `color` 控制 | 控制栏、按钮等小图标 |
| SvgIcon 组件 | 使用项目统一 SvgIcon，通过 `theme` 控制颜色 | 菜单、列表等需要多种语义色的场景 |
| CSS 变量 | 直接 `stroke: var(--td-text-color-secondary)` | 复杂 SVG 或无法内联时 |

### 2.2 颜色变量映射

| 状态 | 变量 | 说明 |
|------|------|------|
| 默认 | `var(--td-text-color-secondary)` | 次要文字色，浅色/深色均适配 |
| 悬停 | `var(--td-text-color-primary)` | 主要文字色 |
| 激活/强调 | `var(--td-brand-color)` | 品牌色 |

---

## 三、已适配图标示例

### 3.1 Input-field 中的 @ 知识库按钮（at-icon）

**修改前**：

```html
<img :src="getImgSrc('at-icon.svg')" alt="@" class="control-icon" />
```

`at-icon.svg` 内部使用 `stroke="#00000099"`，两主题下均为黑色。

**修改后**：

```html
<svg
  width="18" height="18" viewBox="0 0 20 20"
  fill="none" class="control-icon at-icon"
  aria-label="@"
>
  <circle cx="10" cy="10" r="3.5" stroke="currentColor" stroke-width="1.8" fill="none" />
  <path d="M13.5 10V11.5C13.5 12.163..." stroke="currentColor" stroke-width="1.8" stroke-linecap="round" stroke-linejoin="round" fill="none" />
</svg>
```

**配套样式**：

```less
.kb-btn .at-icon {
  color: var(--td-text-color-secondary, #666);
}
.kb-btn:not(.active):not(.disabled):hover .at-icon {
  color: var(--td-text-color-primary, #333);
}
.kb-btn.active .at-icon {
  color: var(--td-brand-color);
}
```

### 3.2 同组件内已有参考

- **websearch-icon**：内联 SVG，`stroke="currentColor"`，由 `.websearch-btn` 的 color 控制
- **model-dropdown-arrow**：`fill="currentColor"`
- **stop-btn**：`fill="currentColor"`

1. 扫描其余 `getImgSrc` 引用的 SVG，优先替换为内联 + `currentColor` 或 SvgIcon

3. 参考 [2026-02-12-自定义SVG组件化方案](./2026-02-12-自定义SVG组件化方案.md) 中图标命名与主题色规范
