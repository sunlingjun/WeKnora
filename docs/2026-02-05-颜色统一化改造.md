## 二、颜色映射对照表

### 2.1 文本颜色
- `#000000e6` / `#111827` / `#333` / `#1d2129` → `var(--td-text-color-primary)`
- `#6B7280` / `#666` / `#86909c` → `var(--td-text-color-secondary)`
- `#9CA3AF` / `#999` / `#00000066` / `#c9ced6` → `var(--td-text-color-placeholder)`
- `#D1D5DB` → `var(--td-text-color-disabled)`
- `#fff` → `var(--td-text-color-anti)`

### 2.2 背景颜色
- `#fff` / `#ffffff` → `var(--td-bg-color-container)`
- `#f5f5f5` / `#fafafa` / `#F9FAFB` / `#f3f4f6` / `#f0f0f0` → `var(--td-bg-color-secondarycontainer)`
- `#f0fdf4` → `var(--td-brand-color-light)` / `#e0f9ea` → `var(--td-success-color-light)`
- `#fff1f0` / `#FEE2E2` → `var(--td-error-color-light)`
- `#FFFBEB` / `#FEF3C7` → `var(--td-warning-color-light)`

### 2.3 边框颜色
- `#e5e5e5` / `#e5e7eb` / `#E7E7E7` / `#d9d9d9` / `#e7ebf0` → `var(--td-component-border)`
- `#d0d0d0` → `var(--td-gray-color-5)`

### 2.4 品牌色与功能色
- `#07C05F` / `#07c05f` → `var(--td-brand-color)`
- `#05a04f` / `#059669` → `var(--td-success-color-6)` / `var(--td-brand-color-active)`
- `#3B82F6` → `var(--td-info-color)`
- `#F59E0B` / `#E37318` / `#92400E` → `var(--td-warning-color-5/6/7)`
- `#fa5151` / `#EF4444` → `var(--td-error-color)`
- `#0052d9` → `var(--td-info-color)`

### 2.5 透明度颜色
- `rgba(0, 0, 0, 0.5)` → `rgba(from var(--td-text-color-primary) r g b / 0.5)`
- `rgba(0, 0, 0, 0.06)` → `rgba(from var(--td-text-color-primary) r g b / 0.06)`
- `rgba(7, 192, 95, 0.12)` → `rgba(from var(--td-brand-color) r g b / 0.12)`

### 2.6 使用示例
```css
/* 替换前 */
.menu-item { color: #07c05f; background: #fff; border: 1px solid #e5e7eb; }

/* 替换后（带 fallback） */
.menu-item {
  color: var(--td-brand-color, #07c05f);
  background: var(--td-bg-color-container, #fff);
  border: 1px solid var(--td-component-border, #e5e7eb);
}
```

## 五、注意事项

1. **始终提供 fallback**：`color: var(--td-brand-color, #07c05f)`
2. **暗色模式**：使用 `--td-text-color-*` 等语义变量，避免直接用 `--td-font-gray-*`
3. **TDesign 覆盖**：`:deep()` 覆盖样式中的颜色也需使用 CSS 变量
